ANOVA

Analiza wariancji.

Opcje​

  • data | object (required): przedmiot tablice wartości. Default: none.
  • variable | string (required): nazwa zmiennej, która ma być wyświetlana. Default: none.
  • group | (string|Factor): nazwa zmiennej grupującej. Default: none.
  • showDecision | boolean: kontroluje, czy należy wyświetlić decyzję o badaniu. Default: false.

Przykłady​

Edytor na żywo
<Anova
    data={heartdisease} 
    variable="Cost"
    group="Drugs"
/>
