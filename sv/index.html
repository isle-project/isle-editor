Author Lessons for the Integrated Statistics Learning Environment

Kom igång

Välkommen till dokumentationen av interaktiv lärandemiljö för statistik (ISLE) och ISLE Editor för att skapa interaktiva lektioner för statistik och relaterade discipliner. Ett av våra mål är att uppmuntra lärare vid universitet och andra utbildningsinstitutioner att använda den för sina ändamål. Vid frågor kan du kontakta oss via e-post på pgb AT andrew.cmu.edu.

Videohandledning

Ett urval av videohandledningar hjälper dig att komma igång och se hur ISLE kan integreras i ditt klassrum.
